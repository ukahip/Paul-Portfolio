About

I'm Ukah Paul Ikechukwu, a Cloud & Security Engineer based in Lagos, Nigeria. I'm currently completing my Diploma in Cloud Engineering at AltSchool Africa, where I've specialised in cloud security and infrastructure design on AWS.

During my internship under Victor Akinode, I deepened my expertise in securing cloud workloads, implementing zero-trust principles, and building resilient AWS-backed systems. That experience shaped how I think about infrastructure — security isn't a feature, it's a foundation.

As my capstone project, I built FoxShare — a secure file vault powered by AWS S3, KMS, Lambda, API Gateway, and Cognito MFA. It represents everything I've learned about building systems that are both functional and genuinely secure.
